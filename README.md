### 功能

---

#### 关于在部署有系统还原、影子系统设备上的安装。
由于权限问题，课表文件存储在 C 盘的用户文件夹内，导致重启后课表还原。
我们不建议使用该类系统，如果您的确需要使用，请先安装ClassTable后再部署相关程序。
或者，您可以等待之后推出的 LTS （Long Term Support）版本，该版本将不在 C 盘中存储课表。
